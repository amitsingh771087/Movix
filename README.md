# Movix Clone

## Live Demo
You can view the live demo of the Movix clone [here](https://movix-dun-nine.vercel.app/).

## Description
Movix Clone is a movie browsing application built with React and Redux. It provides a comprehensive platform for exploring movies, including searching for specific titles, viewing detailed information about movies, and checking user reviews and ratings.

## Features
- **Movie Listings:** Browse through a variety of movies with detailed information.
- **Search Functionality:** Find movies quickly by searching for specific titles.
- **Detailed Movie Information:** View detailed information including summaries, ratings, and more.
- **User Reviews:** Check out user reviews and ratings for movies.

## Tech Stack
- **Frontend:** React, Redux, HTML, CSS, JavaScript
- **API:** TMDB (The Movie Database) [Note: TMDB has been banned by the Government of India for security reasons, which may affect some functionalities.]
